# RepoRoast
RepoRoast is a web-based developer tool that ingests any public GitHub repository and runs it through a panel of three AI judge agents who debate the code in real time. The debate streams live to a split-screen UI, with each judge arguing from a distinct persona — innovation critic, senior architect, and impact assessor.

The output is a structured Judge Panel Report scoring the repo across all four GenAI Genesis judging criteria: **Innovation, Technical Complexity, Product Experience, and Impact***. Every team at the hackathon can use it tonight to stress-test their own project before real judging.

The aesthetic is dark-terminal meets fighting game HUD — serious enough for technical judges, visually exciting enough for vibe coders and a live demo.
